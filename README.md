# Data Classification Using AI

## Project Overview
This project is a basic Artificial Intelligence classification model developed using Python and Machine Learning. The objective of this project is to train an AI model to classify data using supervised learning techniques.

In this project, the Iris dataset is used to train and test the model. The model learns patterns from the dataset and predicts categories based on input features.

## Objective
The main objective of this project is to:

- Load and understand a dataset
- Split data into training and testing sets
- Train an AI classification model
- Test model performance
- Measure prediction accuracy

## Technologies Used
- Python
- VS Code
- Pandas
- Scikit-learn
- Matplotlib

## Dataset Used
The project uses the **Iris Dataset**, a well-known dataset in Machine Learning.

The dataset contains:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The AI model predicts the type of flower.

## Algorithm Used
**Decision Tree Classifier**

A Decision Tree Classifier is a supervised machine learning algorithm used for classification tasks. It learns from training data and predicts categories for new data.

## Project Workflow
1. Import required libraries
2. Load the dataset
3. Separate features and target labels
4. Split data into training and testing sets
5. Train the AI model
6. Make predictions
7. Calculate accuracy
